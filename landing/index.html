🔐

Certificate Intelligence

Upload a PEM or CRT file — Tribal extracts the expiry date automatically. Or point it at a live TLS endpoint to fetch the cert directly.

📋

Full Audit Trail

Every create, update, delete, cert upload, login, and role change is logged with the acting user, timestamp, and source (UI or API).

🔔

Proactive Reminders

Automatic Slack alerts at configurable intervals before expiration, with rotation instructions included so your team can act immediately.

🔄

Safe Deletion

Deleted resources are soft-deleted and recoverable from the Admin panel. Deletion events notify both the owning team and admins via Slack.

👥

Role-based Access

Three-tier access: admins control notification policies and user management, members manage resources, and read-only users get full visibility without edit access.

🔑

API & Automation

REST API with API key authentication. All API activity is attributed in the audit log — distinguish UI-driven changes from automated pipelines.

🏗️

Terraform Provider

Declare and version-control your credential policy alongside your infrastructure. Manage expiration dates and rotation instructions as code.

📅

Visual Calendar

See everything expiring at a glance in month or year view. Overdue and urgent items are highlighted so nothing is missed.

📄

Resource Reports

Generate a PDF snapshot of any resource — full details, ownership history, and a filtered audit log of every change — available to all users regardless of role.
