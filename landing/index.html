🔐

Certificate Intelligence

Upload PEM files or enter a TLS endpoint URL. Tribal fetches the live certificate and automatically extracts the expiration date, keeping your records accurate without manual updates.

🔑

API Key Auto-Detection

Paste any API key and Tribal identifies the provider (GitHub, Stripe, AWS, Azure, and more), looks up the expiry if available, and pre-fills rotation instructions automatically.

📋

Full Audit Trail

Every action is logged — creations, updates, deletions, logins, and role changes. Admins can review the complete history with filtering and pagination.

🔔

Proactive Reminders

Configure Slack notifications at 1, 3, 7, 14, 30, 45, or 60 days before expiry. Per-resource webhooks let different teams get notified in the right channels.

🗑️

Safe Deletion

Deleted resources are soft-deleted and recoverable. Admins can restore or permanently purge them from the deleted resources panel.

👥

Role-Based Access

Three roles: Admin (full control), Member (create and manage resources), and Read-only (view only). The account creator cannot be demoted.

🤖

API & Automation

Full REST API with Bearer token authentication. Create and manage resources programmatically with the same capabilities as the UI.

🏗️

Terraform Provider

Manage credentials as infrastructure-as-code using the seaburr/tribal Terraform provider. Track resources alongside the infrastructure that uses them.

📅

Visual Calendar

See all upcoming expirations on a monthly calendar view. Click any date to see what expires that day. Toggle to a year-at-a-glance view.

📄

Resource Reports

Generate PDF snapshots of any resource including full details and filtered audit history. Useful for compliance reviews and handoffs.

🔌

Extensible Providers

Built-in detection for GitHub, Stripe, AWS, Azure, OpenAI, Anthropic, Slack, and SendGrid. More providers added regularly.
